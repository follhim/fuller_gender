0.2 Present Study

While gender differences have been examined across a host of disciplines, few have examined future religious leaders. Our current studies examine psychological, spiritual, and character differences in men and women.

Spiritual: Spiritual discipline, scripture, spiritual struggle, prayer, self-control, attachement to God anxiety, attachment to God vulnerability

Character: Forgiveness of others, forgiveness of self, compassion, humility

Personality: Narcissism Vulnerability, Narcissism Grandiosity, Authoritarian, Christian nationalism

Psychological: Emotional suppression, differentiatin of self, avoidance

Distress: Burnout-Work, Burnout-personal, anxiety symptoms,

For the current study we will examine differences across several variables using t-tests.

2 Analyses

Code 
perform_t_test <- function(variable, group, data) {
  model <- t.test(reformulate(group, response = variable), data = data)
  results <- tidy(model, conf.int = FALSE)[c("estimate", "estimate1", "estimate2", "statistic", "p.value")]
  results$variable <- variable
  return(results)
}

variables <- c("SPDS6r", "SCRI6r", "SSG6r", "PRAY6r", "SCON6r", "ATGAV6r", "ATGAX6r") # for example
group <- "gender6"

# Apply the function to each variable and combine results
results <- lapply(variables, function(v) perform_t_test(v, group, data))
combined_results <- bind_rows(results)

# Rearrange the columns to have the variable name first and rename columns
final_results <- combined_results %>% 
  select(variable, everything()) %>%
  rename("M1-M2" = estimate, "Mean 1" = estimate1, "Mean 2" = estimate2)

# Round the numeric columns to three decimal places
final_results_rounded <- final_results %>%
  mutate(across(where(is.numeric), ~ round(., digits = 3)))

# Display the final rounded results using gt
final_results <- gt::gt(final_results_rounded) %>%
  tab_header(
    title = "t-Test Results"
  )
